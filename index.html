Home Commands Contact
add-existing-player
Este comando es usado para agregar a un jugador que ya existe en los registros del bot (véase check-player) a un equipo.

Este comando solo puede ser usado por un líder de equipo.
change-team-clantag
Este comando permite cambiar el clantag de un equipo registrado. Solo puede ser usado por un líder de equipo.
change-teammate-clantag
Este comando permite cambiar el clantag de un compañero de equipo. Solo puede ser usado por un líder de equipo.
delete-team-registry
Este comando permite eliminar el registro de un equipo. Solo puede ser usado por un administrador.
delete-teammate-registry
Este comando permite eliminar el registro de un compañero de equipo. Solo puede ser usado por un administrador.
personal-stats-tier10
Este comando permite ver las estadísticas personales de un jugador en el nivel 10. Solo puede ser usado por el jugador.
roster
Este comando muestra la lista de jugadores registrados en un equipo.
single-teammate-registration
Este comando permite registrar a un solo compañero de equipo. Solo puede ser usado por un líder de equipo.
team-registration
Este comando permite registrar un equipo completo. Solo puede ser usado por un líder de equipo.
team-stats
Este comando muestra las estadísticas del equipo registrado.
